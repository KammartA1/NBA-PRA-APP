Risk Management

Bankroll Governance Framework

Fractional Kelly Sizing

Position sizes derived from edge magnitude and confidence interval. Never risks more than the Kelly-optimal fraction of bankroll on any single entry.

Typical: 0.25x - 0.5x Kelly

Drawdown Controls

  • Daily loss limit halts all new entries
  • Weekly drawdown cap triggers reduced sizing
  • Streak-based kill switch after N consecutive losses
  • Session-level bankroll floor enforced

Regime-Based Exposure

  • SHARP -- Full Kelly fraction permitted
  • NORMAL -- Standard sizing applied
  • VOLATILE -- Reduced to 50% max size
  • CHAOTIC -- Minimal exposure only
  • DEAD -- Zero allocation, no entries

Kill Switches

  • CLV degradation below threshold
  • Model confidence interval blowout
  • Data staleness or API failure
  • Manual override always available
